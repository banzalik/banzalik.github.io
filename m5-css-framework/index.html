M5 CSS framework

  • 29 Oct 2008
  • By Александр Бойченко
  • m5 css framework

Что это такое?

M5 CSS framework – это набор css правил, облегчающих разработку сайта. Основная польза от него – это вполне приемлемая типографика.

Я принципиально не хочу вносить в него стили для наброски макета, такие как – шапка, подвал, меню и тд и тп, оставляя полную свободу разработчику.

Последнюю версию можно посмотреть по адресу  banzalik.ru/loremipsum, или-же скачать с m5css.zip

M5 CSS основан на Tripoli, мною был переработан и дополнен под конкретные задачи.

Основные отличия от Tripoli:

  • исправлен баг с password полями
  • добавлен clearfix
  • убрана локализация под разные языки (оформление цитат)
  • у body по умолчанию белый цвет
  • изменены некоторые базовые шрифты

p.s. M5 CSS framework использует компания media5 на своих проектах
